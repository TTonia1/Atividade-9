O form deve ser definido por method e por action

Action

Ele é para onde envia os dados para outro arquivo

Method

O metodo get usado para dados não seguros, usado para que o servidor envie de volta um determinado recurso

O metodo post, os dados são enviados ao servidor, e é feita uma vereficação e retorna um resultado adequado

Tag input

Ela deve estar dentro do container form

Atributo type, define como a tag input ira se comportar, pois possui valores pre definidos

Receber os dados do formulario

O post é a maneira em que os dados do formulario são enviados e podem ser manipulados por PHP
